# Choosing Items

## findInventoryItem
Using [window.findInventoryItem](/windows/api#windowfindinventoryitem) is the easiest method to get an item from your **inventory**.
```js copy
window.findInventoryItem('stone');
bot.inventory.findInventoryItem('stone'); // bot.inventory is also a window
```

## findContainerItem
Using [window.findContainerItem](/windows/api#windowfindcontaineritem) is the easiest method to get an item from a **container**.
```js copy
window.findContainerItem('stone');
```
